# Repair Design Project

## О проекте

**Repair Design Project** — это веб-сайт, предлагающий услуги по ремонту "под ключ" в городе Ростов-на-Дону. Проект демонстрирует различные типы ремонта, предоставляет информацию о завершенных проектах.

## Используемые технологии

- **HTML5** для создания структуры страниц.
- **CSS3** для стилизации и адаптивного дизайна.

## Как запустить проект

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/LidaMudota/repair-project.git
    ```

2. Перейдите в папку проекта:
    ```bash
    cd repair-project
    ```

3. Откройте файл `index.html` в вашем браузере.
